# Плагины источников

Здесь будут плагины источников игр. Каждый каталог — отдельный источник со своим пайплайном установки и обновления.

## Планируемые

| Плагин | Особенности установки |
|--------|------------------------|
| `online-fix` | Portable-архивы, распаковка |
| `freetp` | Installer / portable / встроенный фикс / отдельный патч |

## Контракт

См. [docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md).

Ядро загружает плагин по манифесту, UI и библиотека работают через единый Core API.
